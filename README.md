A comprehensive cybersecurity vulnerability scanner with web interface built with Python Flask.

## Features

- **Port Scanner**: Scan for open ports and identify services
- **SQL Injection Detector**: Test web applications for SQL injection vulnerabilities
- **XSS Scanner**: Detect Cross-Site Scripting vulnerabilities
- **Password Strength Analyzer**: Check password security and strength
- **SSL/TLS Checker**: Verify SSL certificates and configuration
- **Web Dashboard**: View scan history and results
- **Report Generation**: Export results in multiple formats
